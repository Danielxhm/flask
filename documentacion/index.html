Introduction

Quick Start Guide for MotoRental API

MotoRental API is a versatile solution that integrates seamlessly with any modern web framework or library. Get up and running in minutes.

Getting Started

Follow these straightforward steps within your preferred framework to begin using the MotoRental API.

Need help deciding which framework to choose?

The team at MotoRental recommends Next.js or Remix for their robust features and server-side capabilities. However, our API is designed to work efficiently with any framework that supports standard HTTP requests.

Step 1: Install Client Library

Run the following command in your terminal to install the MotoRental client package:

npm install @motorental/client-sdk
# or using yarn
yarn add @motorental/client-sdk

Step 2: Activate Your API Key (Optional)

If you have a premium plan, initialize the client with your API key for extended features:

your-app.js 
import { MotoRentalClient } from '@motorental/client-sdk';

const apiKey = 'YOUR_API_KEY_HERE';
const client = new MotoRentalClient({ apiKey });

// Example: Fetch available motorcycles
async function getAvailableMotos() {
  try {
    const motos = await client.getMotos({ availability: 'available' });
    console.log('Available Motorcycles:', motos);
  } catch (error) {
    console.error('Error fetching motos:', error);
  }
}

getAvailableMotos();
